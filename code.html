11.23 | "site to neocities profile" bookmarklet A bookmarklet looks like a regular bookmark but runs javascript on the current page when clicked rather than taking you to a new one. This bookmarklet reorders urls like example.neocities.org/randompage to neocities.org/site/example when pressed. to use: - make a new bookmark of any page - right click it > edit, then replace the url field with the following code:

javascript: (() => { window.location = 'https://neocities.org/site/' + window.location.hostname.split('.')[0]; })();

you can change the title of the bookmark to anything--it has no bearing on functionality.
