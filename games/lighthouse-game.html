unnamed lighthouse game

developed ~june 2022 This was a first-person puzzle game where the goal was to light up the lighthouse. The idea was to scale up an escape room and have puzzles in a few places over the island. I got pretty sidetracked trying to learn to write shaders (trying to get some not really relevant thing to work, classic noob trap). This was half to just try and learn some stuff about unity so it was successful in that respect, I guess. reason for dropping: I learned I did not like making 'realistic' assets, they felt like such a low return for the time investment required. Also, each puzzle would require custom programming to keep the challenges different and I'm just not interested in creating that much work for myself. likelihood of revival: Since the game was a heavily content-oriented design this thing is dead dead.
